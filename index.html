Guía Completa de Funciones Utilizadas en los Ejercicios

1. printf

Descripción: Se utiliza para imprimir texto o valores en la consola.

Sintaxis:

printf("Texto con formato", valores);

Ejemplo:

int numero = 10;
printf("El número es: %d\n", numero);

Cuándo usarla: Siempre que necesites mostrar información al usuario.

2. scanf

Descripción: Se utiliza para leer datos desde la entrada estándar (teclado).

Sintaxis:

scanf("Formato", &variable);

Ejemplo:

int numero;
printf("Ingrese un número: ");
scanf("%d", &numero);

Cuándo usarla: Cuando necesites capturar datos ingresados por el usuario.

3. fgets

Descripción: Se utiliza para leer una línea completa de texto, incluyendo espacios.

Sintaxis:

fgets(buffer, tamaño, stdin);

Ejemplo:

char cadena[100];
printf("Ingrese una cadena: ");
fgets(cadena, 100, stdin);

Cuándo usarla: Cuando necesites leer cadenas de texto con espacios.

4. strstr

Descripción: Busca una subcadena dentro de otra cadena.

Sintaxis:

const char *resultado = strstr(cadena, subcadena);

Ejemplo:

const char *cadena = "palanca";
const char *subcadena = "pala";
if (strstr(cadena, subcadena)) {
    printf("La subcadena está contenida.\n");
}

Cuándo usarla: Para verificar si una cadena está contenida dentro de otra.

5. strlen

Descripción: Calcula la longitud de una cadena (sin incluir el carácter nulo '\0').

Sintaxis:

size_t longitud = strlen(cadena);

Ejemplo:

const char *cadena = "Hola";
printf("La longitud de la cadena es: %lu\n", strlen(cadena));

Cuándo usarla: Para obtener el tamaño de una cadena.

6. toupper

Descripción: Convierte un carácter a mayúscula.

Sintaxis:

char mayuscula = toupper(caracter);

Ejemplo:

char letra = 'a';
printf("Letra en mayúscula: %c\n", toupper(letra));

Cuándo usarla: Para comparar cadenas ignorando diferencias entre mayúsculas y minúsculas.

7. rand y srand

Descripción: Se utilizan para generar números pseudoaleatorios.

Sintaxis:

srand(time(NULL)); // Inicializa la semilla
int numero = rand(); // Genera un número aleatorio

Ejemplo:

srand(time(NULL));
int aleatorio = rand() % 100; // Número entre 0 y 99
printf("Número aleatorio: %d\n", aleatorio);

Cuándo usarla: Para generar números aleatorios en tus programas.

8. strcmp

Descripción: Compara dos cadenas de texto.

Sintaxis:

int resultado = strcmp(cadena1, cadena2);

Ejemplo:

const char *cadena1 = "Hola";
const char *cadena2 = "Mundo";
if (strcmp(cadena1, cadena2) == 0) {
    printf("Las cadenas son iguales.\n");
} else {
    printf("Las cadenas son diferentes.\n");
}

Cuándo usarla: Para comparar si dos cadenas son iguales o determinar su orden lexicográfico.

9. strcpy

Descripción: Copia el contenido de una cadena a otra.

Sintaxis:

strcpy(destino, origen);

Ejemplo:

char destino[50];
const char *origen = "Hola Mundo";
strcpy(destino, origen);
printf("Destino: %s\n", destino);

Cuándo usarla: Para duplicar o copiar cadenas de texto.

10. strcat

Descripción: Concatena (une) dos cadenas.

Sintaxis:

strcat(destino, origen);

Ejemplo:

char destino[50] = "Hola ";
const char *origen = "Mundo";
strcat(destino, origen);
printf("Resultado: %s\n", destino);

Cuándo usarla: Para unir dos cadenas en una sola.

11. memset

Descripción: Inicializa un bloque de memoria con un valor específico.

Sintaxis:

memset(ptr, valor, tamaño);

Ejemplo:

char buffer[50];
memset(buffer, 0, sizeof(buffer)); // Inicializa el buffer con ceros

Cuándo usarla: Para limpiar o inicializar bloques de memoria.

12. free y malloc

Descripción: Se utilizan para gestionar memoria dinámica.

Sintaxis:

ptr = malloc(tamaño);
free(ptr);

Ejemplo:

int *array = (int *)malloc(5 * sizeof(int));
if (array == NULL) {
    printf("Error al asignar memoria.\n");
    return 1;
}
array[0] = 10;
printf("Primer elemento: %d\n", array[0]);
free(array);

Cuándo usarla: Para asignar y liberar memoria dinámica.

13. exit

Descripción: Termina la ejecución del programa inmediatamente.

Sintaxis:

exit(código);

Ejemplo:

if (error) {
    printf("Ocurrió un error.\n");
    exit(1);
}

Cuándo usarla: Para salir del programa en caso de error o condiciones específicas.

14. Funciones personalizadas

Descripción: Permiten encapsular lógica reutilizable.

Sintaxis:

tipo_de_retorno nombre_función(tipo_parametro nombre_parametro, ...) {
    // Código de la función
    return valor;
}

Ejemplo:

double average(double marks[], int size) {
    double sum = 0;
    for (int i = 0; i < size; i++) {
        sum += marks[i];
    }
    return sum / size;
}

Cuándo usarla: Cuando necesites dividir tu programa en bloques reutilizables.

15. Librerías comunes

  • <stdio.h>: Para entrada y salida estándar (printf, scanf).
  • <stdlib.h>: Para funciones como rand y malloc.
  • <string.h>: Para manipulación de cadenas (strlen, strstr, strcmp, strcpy, strcat).
  • <ctype.h>: Para manipulación de caracteres (toupper).
  • <time.h>: Para trabajar con tiempo y generar números aleatorios.
