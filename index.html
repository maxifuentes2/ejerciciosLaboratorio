Apunte Teórico sobre Conceptos Utilizados en C

1. Estructuras de Control

a. Bucle for

Se utiliza para repetir un bloque de código un número fijo de veces.
Sintaxis: 

for (inicialización; condición; incremento) {
    // Código a ejecutar
}

Ejemplo:

// Imprimir los números del 1 al 5
for (int i = 1; i <= 5; i++) {
    printf("%d\n", i);
}

b. Bucle while

Se utiliza para repetir un bloque de código mientras una condición sea verdadera.
Sintaxis: 

while (condición) {
    // Código a ejecutar
}

Ejemplo:

// Imprimir los números del 1 al 5
int i = 1;
while (i <= 5) {
    printf("%d\n", i);
    i++;
}

c. Condicional if

Se utiliza para tomar decisiones basadas en una condición.
Sintaxis: 

if (condición) {
    // Código si la condición es verdadera
} else {
    // Código si la condición es falsa
}

Ejemplo:

// Verificar si un número es positivo o negativo
int numero = -5;
if (numero > 0) {
    printf("El número es positivo.");
} else {
    printf("El número es negativo.");
}

2. Funciones

Las funciones encapsulan lógica reutilizable y permiten dividir el código en bloques más manejables.
Sintaxis: 

tipo_de_retorno nombre_función(tipo_parametro nombre_parametro, ...) {
    // Código de la función
    return valor;
}

Ejemplo:

// Función para calcular el promedio de un arreglo
double average(double marks[], int size) {
    double sum = 0;
    for (int i = 0; i < size; i++) {
        sum += marks[i];
    }
    return sum / size;
}

3. Arreglos

Los arreglos son estructuras que almacenan múltiples valores del mismo tipo en posiciones consecutivas de memoria.
Sintaxis: tipo nombre_arreglo[tamaño];

Ejemplo:

// Crear un arreglo de 5 enteros
int arreglo[5] = {1, 2, 3, 4, 5};
printf("El primer elemento es: %d\n", arreglo[0]);

4. Cadenas de Texto

Las cadenas son arreglos de caracteres terminados en '\0'.
Funciones comunes:

  • strlen: Devuelve la longitud de una cadena.
  • strstr: Busca una subcadena dentro de otra.
  • toupper: Convierte un carácter a mayúscula.

Ejemplo:

char cadena[] = "Hola";
printf("La longitud de la cadena es: %lu\n", strlen(cadena));

5. Generación de Números Aleatorios

Se utiliza rand() para generar números pseudoaleatorios.
Para inicializar la semilla, se usa srand(time(NULL)).

Ejemplo:

srand(time(NULL));
int numero = rand() % 100; // Número aleatorio entre 0 y 99
printf("Número aleatorio: %d\n", numero);

6. Constantes y Defines

Se utilizan para definir valores constantes que no cambian durante la ejecución del programa.
Sintaxis: #define NOMBRE_VALOR valor

Ejemplo:

#define MAX_GOLES 6
printf("El máximo de goles es: %d\n", MAX_GOLES);

7. Tipos de Datos

- int: Enteros (e.g., 1, 2, 3).
- float y double: Números con decimales (e.g., 3.14, 2.718).
- char: Caracteres (e.g., 'a', 'b').
- bool: Valores booleanos (requiere <stdbool.h>).

8. Entrada y Salida

printf: Imprime texto en la consola.
scanf: Lee datos desde la entrada estándar.

Ejemplo:

int numero;
printf("Ingrese un número: ");
scanf("%d", &numero);
printf("El número ingresado es: %d\n", numero);

9. Manejo de Matrices

Las matrices son arreglos bidimensionales.
Sintaxis: tipo nombre_matriz[filas][columnas];

Ejemplo:

int matriz[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};
printf("El elemento en la posición [1][2] es: %d\n", matriz[1][2]);

10. Operadores

- Aritméticos: +, -, *, /, %.
- Relacionales: ==, !=, <, >, <=, >=.
- Lógicos: &&, ||, !.

Ejemplo:

int a = 5, b = 10;
if (a < b && b > 0) {
    printf("a es menor que b y b es positivo.\n");
}

11. Librerías Utilizadas

- <stdio.h>: Para entrada y salida estándar.
- <stdlib.h>: Para funciones como rand y srand.
- <string.h>: Para manipulación de cadenas.
- <ctype.h>: Para funciones de manipulación de caracteres.
- <stdbool.h>: Para trabajar con valores booleanos.
- <time.h>: Para trabajar con tiempo y generar números aleatorios.

12. Ejemplos Prácticos

Promedio de un arreglo:

double notas[] = {7.5, 8.0, 6.5};
int cantidad = 3;
double promedio = average(notas, cantidad);
printf("El promedio es: %.2f\n", promedio);

Comparar vectores en orden relativo:

char arreglo1[] = {'A', 'B', 'C'};
char arreglo2[] = {'C', 'A', 'B'};
if (sonIguales(arreglo1, arreglo2, 3)) {
    printf("Los vectores son iguales.\n");
} else {
    printf("Los vectores no son iguales.\n");
}

Determinar si una cadena es palíndromo:

char palabra[] = "NEUQUEN";
if (esPalindromo(palabra)) {
    printf("La palabra es un palíndromo.\n");
} else {
    printf("La palabra no es un palíndromo.\n");
}
